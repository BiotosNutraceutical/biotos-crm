Biotos CRM – v6 (PWA, mobile-first)
====================================
Build: 2025-09-06

Novità chiave
- **Mobile-first**: liste in **card** facili da leggere; pulsante **＋ flottante** per aggiungere rapidamente; **sheet** a tutta larghezza per creare/modificare medici, farmacie e visite.
- **Desktop ottimizzato**: tabelle classiche con bottoni Modifica/Elimina; top-nav al posto della bottom-nav.
- **Persistenza**: IndexedDB + LocalStorage; indicatori “Salvando…” / “Salvato ✓”.
- **Follow-up**: anche in card su mobile; completamento con un tap.
- **Accessibilità & touch**: target 44px, contrasto aumentato, momentum scrolling.

Come pubblicare
1) Carica questi file su GitHub Pages.
2) Apri l’URL con Safari (iPhone) → Aggiungi a Home.
3) Se avevi versioni vecchie: chiudi e riapri l’app per attivare la cache `biotos-crm-v6`.

Backup/Restore
- JSON (backup completo) nella tab **Dati**.
- CSV per Medici/Farmacie/Visite nella rispettiva sezione.
